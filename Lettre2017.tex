Vous connaissez bien notre logiciel   {\sffamily\bfseries
Apprenti Géomètre}, un logiciel qui  fournit un éclairage
dynamique à la géométrie, de l'école primaire à la fin du
secondaire. Une nouvelle version  vous attend sur notre site
internet www.crem.be. Cette nouvelle version, notée 2.5.0,
comporte cinq variantes:

\begin{itemize} \item Une variante 32 bits et une variante 64
bits pour le travail sur pc Windows.  \item Une variante 32 bits
et une variante 64 bits pour le travail sous Linux.
\item Une variante 32 bits pour le travail sous Macintosh.
\end{itemize}

Toutes ces variantes sont téléchargeables gratuitement.

Signalons
une nouveauté qui rendra vos figures plus attrayantes:  la
possibilité de rendre semi-transparente la couleur de fond
d'une forme géométrique. Fini les formes qui se cachent les-unes
les-autres, partiellement ou totalement!\end{document}
